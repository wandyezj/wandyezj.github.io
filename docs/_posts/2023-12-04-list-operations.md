## Initialize

Initialize list and assign to variable.

Python

```python
# empty list
my_list = []

# list with 3 elements
my_list = [1, 2, 3] 
```

JavaScript

```javascript
const my_list = [1,2,3];
```

## Access Element

Index element in list and assign to variable.

Python

```python
element = my_list[0] # first element 1
```

JavaScript

```javascript
my_list[0]; // first element 1
```


## Change Element at Index

Python

```python
# change first element to 100
my_list[0] = 100
```

JavaScript

```javascript
my_list[0] = 100;
```

## Add Element to End

Python

```python
# add element to end of list
element = 4
my_list.append(element)
```

JavaScript

```javascript
const element = 4;
my_list.push(element);
```

## Remove Element from End

```python
# remove element from end of list
my_list.pop()
```
