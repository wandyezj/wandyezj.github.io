# WSL

## list distros

> wsl --list --verbose

## Update WSL version

> wsl --set-version `distro-name` 2

## Check ubuntu version

> lsb_release -a

> cat /etc/issue

> cat /etc/os-release

> cat /proc/version

# Upgrade Ubuntu Version

> sudo apt update

> sudo apt upgrade

> sudo apt dist-upgrade

> sudo do-release-upgrade
